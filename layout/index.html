单列布局

水平居中

水平居中的页面布局中最为常见的一种布局形式，多出现于标题，以及内容区域的组织形式，下面介绍四种实现水平居中的方法（注：下面各个实例中实现的是child元素的对齐操作，child元素的父容器是parent元素）

1.使用inline-block 和 text-align实现

.parent{text-align: center;}

.child{display: inline-block;}

效果:

水平居中

优点：兼容性好；

不足：需要同时设置子元素和父元素

2.使用margin:0 auto来实现

.child{width:200px;margin:0 auto;}

效果:

水平居中

优点：兼容性好；

缺点: 需要指定宽度

3.使用table实现

.child{display:table; margin: 0 auto;}

效果:

水平居中

优点:只需要对自身进行设置

不足:IE6,7需要调整结构

4.使用绝对定位实现

.parent{position: relative;}

.child{position: absolute; left: 50%; transform: translate(-50%);}

效果:

水平居中

或者使用margin-left的负值为盒子宽度的一半也可以实现，不过这样就必须知道盒子的宽度，但兼容性好

不足：transform兼容性差,IE9及以上可用

5.实用flex布局实现

/*第一种方法*/

.parent{display: flex; justify-content: center;}

效果:

水平居中

/*第二种方法*/

.parent{display: flex;}

.child{margin: 0 auto;}

效果:

水平居中

缺点：兼容性差，如果进行大面积的布局可能会影响效率

垂直居中

下面介绍三种实现垂直居中的方法

1.实用vertical-align实现

/*第一种方法*/

.parent{display: table-cell; vertical-align: middle; height: 50px;}

效果:

垂直居中

/*第二种方法*/

.parent{display: inline-block; vertical-align: middle; line-height: 50px;}

效果:

垂直居中

2.使用绝对定位实现

.parent{position: relative;}

.child{position: absolute; top: 50%; transform: translate(0,-50%);}

效果:

垂直居中

3.实用flex实现

.parent{display:flex;align-items:center;}

效果:

垂直居中

水平垂直居中

下面介绍三种实现水平垂直居中的方法

1.利用vertical-align,text-align,inline-block实现

.parent{display: table-cell; vertical-align: middle; text-align:center;}

.child{display: inline-block;}

效果:

水平垂直居中

2.利用绝对定位实现

.parent{position: relative;}

.child{position: absolute; top: 50%; left: 50%; transform: translate(-50%, -50%); /*或者margin-left: -50%; margin-top: 50%;*/}

效果:

水平垂直居中

3.利用flex实现

.parent{display: flex; justify-content: center; align-items: center;}

效果:

水平垂直居中
